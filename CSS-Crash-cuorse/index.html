CSS (Cascading Style Sheets) is a styling language used to control the appearance and layout of HTML elements on a webpage. It allows developers to apply colors, fonts, spacing, animations, and responsive designs, making websites visually appealing and user-friendly across different devices. CSS works by selecting HTML elements and applying style rules, enabling separation of content from presentation for cleaner, more maintainable code.

style for div

This code is an external stylesheet used for styling HTML pages. In the first part, element selectors (h2 and p) are used to change all level-2 headings to red and all paragraphs to navy blue. Additionally, an ID selector (#para1) targets a specific paragraph with the ID para1 and changes its color to black, which has a higher priority than the element selector. Furthermore, a class selector (.strong-text) applies brown color to any element that has this class, allowing it to be reused across multiple elements. In the second part, the body styling is defined, which includes setting the text direction to left-to-right (ltr), a light gray background, and specifying Segoe UI as the default font with a size of 14 pixels and normal weight. The line height is also set to 1.5 times the font size to improve readability. These base settings create a consistent and professional appearance across the entire page, enhancing the overall user experience.
